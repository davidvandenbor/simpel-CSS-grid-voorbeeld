CSS grid?

Met 1 regeltje in je CSS code (grid-auto-flow:columns) kun je elke groep items in een HTML container veranderen in een grid met columns. Dit is gebruikt voor de kolommen die je hier ziet op deze pagina. maar ook voor het navigatie menu hier rechtsboven! Kijk maar in de CSS code!
